Développement de l'Application Mobile pour FitLife

FitLife est une startup innovante qui propose des programmes de fitness personnalisés. Le projet consistait à développer une application mobile intuitive pour offrir aux utilisateurs une expérience fluide et motivante. Mon rôle comprenait la gestion de projet, la coordination des équipes et l'organisation de tests utilisateurs pour optimiser l'application.

Objectifs :

  • Développer une application intuitive et user-friendly
  • Assurer le respect des délais de développement
  • Améliorer l'expérience utilisateur grâce aux tests

Résultats :

  • Lancement dans les délais prévus
  • Note moyenne de 4.5 étoiles sur l'App Store
  • 10,000 téléchargements dans les 3 premiers mois
